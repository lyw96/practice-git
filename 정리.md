# input()함수
입력하는 함수  
input('입력 : ')하면  
입력 : 이라고 출력되고 입력창이 출력, 입력하면 입력한게 나옴  

# 변수
a = 10  
print(a)  
실행결과 : 10  
a는 10이라 치자~  

예시)  
name = '유리'  
height = 180  
age = 30  
hobby = '요리하기'  
print(name, '키는', height, '나이는', age, '취미는', hobby, '입니다!' )  
실행결과 : 유리 키는 180 나이는 30 취미는 요리하기 입니다!  
print('{} 키는 {}, 나이는 {}, 취미는 {}입니다!' .format(name, height, age, hobby)) 라고 입력헤도 나옴
